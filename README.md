# Buttermilkmicrobiome
The effects of a buttermilk-incorporated diet on modulating gut microbiota is investigated using Qiime2 tools and analysis techniques taught in the class.

The Submit_to_hpc_run3.slurm file contains commands used to generate analysis data from raw experimental data in the metadata.tsv file. Next, I used R files (Alpha_Diversity.R, beta_diversity.R, cooccurrence.R, and taxa_Project_Buttermilk.R) to generate plots shown in the report using the qiime outputs.
